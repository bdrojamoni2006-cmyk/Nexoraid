Firebase Integration Info

NEXORAID comes with fully production-ready Firebase integration. By default, it operates on a beautiful local cache mechanism because its firebaseConfig is set as a placeholder.

How to connect your live DB:

  1. Navigate to /src/firebase.ts or edit inside index.html.
  2. Populate the firebaseConfig credentials with your actual Firebase project settings.
  3. The registry, login systems, and administrator database portal will immediately switch to live synchronization!
System Info: Operates using the compat v9.23.0 SDK module. Users are stored in the users collection inside Firestore.
